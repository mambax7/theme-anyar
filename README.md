Anyar Bootstrap Theme For XOOPS CMS 2.5.8.x
=====

![Theme Preview](/theme_preview.png)

Anyar is a very simple multipurpose one page template,this theme built with responsive Bootstrap 3.3 framework.

The template has been designed for multi purpose use and can be used by creative agencies, individuals, companies looking for a simple and clean one page template.

Ported to XOOPS [CMS 2.5.8.x](http://xoops.org/modules/news/article.php?storyid=6762)
